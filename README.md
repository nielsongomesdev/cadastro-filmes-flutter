## Sobre
Projeto inicial em Flutter que permite cadastrar filmes e visualizá-los em uma lista. Ideal para quem está começando com Flutter/Dart e quer um hands-on em criação de telas, navegação e persistência simples (ou integração com serviços).

## Funcionalidades
- Cadastro de filmes (título, ano, descrição, nota)
- Listagem dos filmes cadastrados
- Visualização dos detalhes do filme
- Exemplo básico de navegação e formulários com validação

## Pré-requisitos
Antes de rodar o projeto, instale:
- Flutter SDK (recomenda-se a última versão estável): https://docs.flutter.dev/get-started/install
- Editor (VS Code, Android Studio ou IntelliJ)
- Emulador Android / dispositivo físico configurado

Confirme a instalação com:
```
flutter --version
flutter doctor
```

## Instalação e execução
1. Clone o repositório:
```
git clone https://github.com/nielsongomesdev/cadastro-filmes-flutter.git
cd cadastro-filmes-flutter
```
2. Instale dependências:
```
flutter pub get
```
3. Rode o app em modo debug:
```
flutter run
```
4. Para gerar build de release (Android):
```
flutter build apk --release
```

## Estrutura do projeto
